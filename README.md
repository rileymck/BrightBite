## Quick Start  
1. **Clone the repository**  
   ```bash
   git clone https://github.com/rileymck/BrightBite.git
   cd brightbite
   ```

2. **Install dependencies**  
   ```bash
   flutter pub get
   ```

3. **Run the app**  
   ```bash
   flutter run
   ```

4. **Design changes?**  
   - Open `figma.com` project link (add yours here)
